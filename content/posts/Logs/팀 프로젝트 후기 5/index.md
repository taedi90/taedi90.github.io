## 2. 구현 Idea

#### 회원가입(일반)

가입신청 → 약관 동의 → 정보입력 → 이메일 확인(~~session?~~ table) → 가입 완료

> 이메일 링크에 아이디를 암호화한 주소를 줘서 해당 주소로 접속하면 확인 되는 걸로 하기  
>   

#### 회원가입(oAuth)

가입신청 → 구글 로그인 및 정보 제공 동의 → (약관 동의) → 가입 완료

> 비밀번호는 암호화 모듈 이용해서 임의로 입력(어차피 미사용하지만 혹시 모를 상황 대비)  
oAuth 속성에 true 주기  
>
